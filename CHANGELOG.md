## 설치 및 실행

### 환경 변수 설정
```bash
# .env 파일
VITE_API_URL=http://localhost:8000
SUPABASE_URL=your_supabase_url
SUPABASE_KEY=your_supabase_key
```

### 개발 환경 실행
```bash
# 백엔드
pip install -r requirements.txt
uvicorn api.main:app --reload

# 프론트엔드
cd frontend
npm install
npm run dev
```
